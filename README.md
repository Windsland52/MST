# MST
基于MaaFramework的灵魂潮汐(SoulTide)日常脚本。

## 功能
- [x] 启动游戏
- [x] 签到(仅支持启动游戏后自动弹出的签到界面)
- [x] 领取邮件、教会补给、商店每日补给
- [ ] 家园
  - [ ] 炼金炉
  - [ ] 种菜收菜
  - [ ] 一键派遣
  - [ ] 捕鱼(渔网)
  - [ ] 拜访箱子
- [ ] 战斗
  - [ ] 副本扫荡
  - [ ] 异象(上榜)
- [ ] 巡查
- [ ] 相伴
  - [ ] 摸头
  - [ ] 送礼
- [ ] 每日抽卡
- [x] 公会每日悬赏
- [ ] 任务奖励
- [ ] 挖矿
