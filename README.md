# MST
基于MaaFramework的灵魂潮汐(SoulTide)日常脚本。

## 功能
- [x] 启动游戏
- [x] 签到(仅支持启动游戏后自动弹出的签到界面)
- [x] 领取邮件、教会补给、商店每日补给
- [x] 家园
  - [x] 炼金炉
  - [x] 种菜收菜
  - [x] 事务派遣
  - [ ] 捕鱼(渔网)
  - [x] 拜访好友
- [x] 战斗
  - [x] 金币本扫荡
  - [x] 异象(上榜)
- [ ] 巡查
- [ ] 相伴
  - [ ] 摸头
  - [ ] 送礼
- [ ] 每日抽卡
- [x] 公会每日悬赏
- [ ] 任务奖励
- [ ] 挖矿
